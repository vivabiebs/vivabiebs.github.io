CSS SPECIFICITY

CSS Specificity เป็นคอนเซปต์การเลือก CSS Selector ที่มี ความสำคัญ เยอะที่สุด มาใช้กับ Element บนหน้าเว็บไซต์

ซึ่ง CSS Selector ได้แก่ element , class , ID และ อีกแบบนึงเลยคือการใส่ style ลงไปใน html เลย

ซึ่งเรียงลำดับความสำคัญได้ดังนี้

1.เขียน CSS ใส่ใน style เลย

2.ID Selector

3.Class Selector

4.Element Selector

จะเห็นได้ว่า ลำดับความสำคัญนี้ สามารถเรียงได้จาก ความจำเพาะเจาะจงไปที่ตัว element นั้นๆ ยิ่งเจาะจงมากเท่าไหร่ CSS จะทำตามคำสั่งนั้นเป็นอันดับแรก

สำหรับ !important ที่ไม่ได้กล่าวถึง ถ้าใส่เข้าไปจะเป็นการบังคับใช้ CSS นั้นทันที ไม่ต้องดูความสำคัญกันเลย

และถ้าหากเราจำเป็นต้องใช้ selector หลายๆอันด้วยกัน ควรทำตามหลักดังนี้

ไม่ควรใช้ #ID ใน Selector – การใช้ ID จะทำให้ความแตกต่างระหว่าง Selector มากเกินไป และเขียนทับได้ยาก (ถ้าจะทับต้องเขียน ID, Inline CSS, หรือ !important เท่านั้น ซึ่งทำให้โค้ดเยอะและยาว)

ไม่ควรเขียน Selector ซ้อนกันโดยไม่จำเป็น – การเขียนซ้อนโดยไม่จำเป็น จะทำให้ค่าความสำคัญสูงกว่าเดิมโดยไม่จำเป็น ยกเว้นว่าแบบแรกไม่ทำงาน

ใช้ Class บ่อยๆ – Class เป็นการเขียน Selector ที่ความสำคัญสูงเกินไป จะเขียนทับก็ไม่ยาก หรือจะสามารถนำมาใช้ใหม่ก็ง่าย
